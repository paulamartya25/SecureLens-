🧠 What is GradCAM and what does this show?
What is GradCAM?
Gradient-weighted Class Activation Mapping (GradCAM) uses the gradients flowing into the last convolutional layer to highlight which image regions most influenced the prediction.
How to read the heatmap
Red/yellow regions = high attention (model focused here). Blue/purple regions = low attention. For pneumonia, expect attention in the lung fields where infiltrates appear.
Why this matters
Explainability proves the model learned medically relevant features — not shortcuts. Radiologists can verify the model is looking at the right regions before trusting its diagnosis.
FHE + GradCAM
The GradCAM runs on the server with the full model. In SecureLens, only the final linear head runs on encrypted data. GradCAM runs as a separate explainability step on demand.
