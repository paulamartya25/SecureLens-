    🧠 What is GradCAM?
    Gradient-weighted Class Activation Mapping flows gradients backward through the final convolutional layer to highlight which image regions most influenced the model's prediction.
    🎨 Reading the heatmap
    Red/yellow = high attention (model focused here). Blue/purple = low attention. For pneumonia, expect attention in lung fields where infiltrates and consolidation appear.
    ✅ Why this matters
    Explainability proves the model learned real medical features — not shortcuts. Radiologists can verify the model focuses on correct anatomical regions before trusting its diagnosis.
    🔒 FHE + GradCAM
    GradCAM runs on the full ResNet model on the server as a separate explainability step. The encrypted FHE inference path uses only the linear head on ciphertexts.
